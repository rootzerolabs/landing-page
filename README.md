# RootZero Labs — landing page

Static site. Just `index.html` + `assets/` (fonts + hero image). No build step.

## Deploy (Vercel, manual)

```
npm i -g vercel   # once
vercel            # preview
vercel --prod     # production
```

Framework preset: **Other**. No build command, output dir is the repo root.
